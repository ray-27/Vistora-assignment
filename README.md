# Vistora Assignment

To run the files you need to create a virtual environment
```bash
python3 -m venv venv
```

then install the required packages
```bash
pip install -r requirement.txt
```

Put the Snowflakes credentials in the `.env` file 
```bash
SNOW_USER=
SNOW_PASS=
SNOW_ACCOUNT=
```

Now you can run the notebooks.
